# 科目列表布局修改计划

## 项目结构分析

通过分析项目代码，我发现科目列表的实现位置如下：

- **HTML文件**：`/Users/ferrisyu/source/SimuExam/templates/index.html` 第98行
  ```html
  <div id="subjectList" class="subject-grid"></div>
  ```

- **CSS样式**：`/Users/ferrisyu/source/SimuExam/templates/css/style.css` 第252-257行
  ```css
  .subject-grid {
      display: flex;
      flex-wrap: wrap;
      gap: 12px;
      margin-top: 16px;
  }
  ```

- **科目项样式**：`/Users/ferrisyu/source/SimuExam/templates/css/style.css` 第259-275行
  ```css
  .subject-item {
      background: #f1f5f9;
      padding: 8px 16px;
      border-radius: 40px;
      display: flex;
      align-items: center;
      gap: 12px;
  }
  ```

## 问题分析

当前科目列表使用了 `flex-wrap: wrap` 的flex布局，导致科目项横向排列并自动换行，看起来比较混乱。用户希望将其改为纵向排列的一列显示。

## 解决方案

### 1. 修改CSS样式

需要修改 `.subject-grid` 类的样式，将其从横向排列改为纵向排列：

```css
.subject-grid {
    display: flex;
    flex-direction: column; /* 改为纵向排列 */
    gap: 8px; /* 调整间距，更适合纵向排列 */
    margin-top: 16px;
}
```

### 2. 调整科目项样式（可选）

为了使纵向排列的科目项看起来更好，可以适当调整 `.subject-item` 的样式：

```css
.subject-item {
    background: #f1f5f9;
    padding: 12px 16px; /* 增加垂直 padding，使项目更醒目 */
    border-radius: 12px; /* 改为圆角矩形，更适合纵向列表 */
    display: flex;
    align-items: center;
    gap: 12px;
}
```
